# Glow & Glamour – Beauty Salon Website

A responsive beauty salon website built with **HTML**, **CSS**, and **JavaScript**. The project showcases a modern beauty salon website with a clean design, responsive layout, and interactive features.

## 🌐 Live Demo

**Website:** https://lainerxs.github.io/Glow-and-Glam/

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Modern and elegant user interface
- Smooth scrolling navigation
- Interactive JavaScript components
- Image gallery
- Contact section

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
